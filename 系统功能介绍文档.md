# 智学方舟 - 学生全周期成长智能规划服务平台

## 1. 系统概述

智学方舟是一个面向大学生的全周期成长智能规划服务平台，旨在帮助学生科学规划学习成长旅程，提供个性化的学习规划和职业发展指导。系统包含学生端、超级管理员端和学校管理员端三个主要部分，通过智能化的功能模块，为学生、学校和管理员提供全方位的服务。

## 2. 系统架构

### 2.1 技术架构

- **前端**：Vue.js 3.x + Element Plus + Vue Router
- **后端**：Spring Boot + MySQL + MyBatis Plus + Spring Security
- **认证**：JWT (JSON Web Token)
- **部署**：前后端分离部署

### 2.2 系统模块

| 模块 | 功能描述 | 角色 |
|------|---------|------|
| 学生端 | 学习规划、资源管理、目标管理、职业规划等 | 学生 |
| 超级管理员端 | 系统管理、用户管理、学校管理等 | 超级管理员 |
| 学校管理员端 | 学校管理、用户管理、资源管理等 | 学校管理员 |

## 6. 系统使用指南

### 6.1 学生用户
1. **注册/登录**：通过学校代码、学号/用户名和密码进行注册和登录
2. **个性化设置**：完善个人信息，设置学习目标和兴趣爱好
3. **使用核心功能**：
   - 制定学习计划
   - 浏览和使用学习资源
   - 设定和跟踪学习目标
   - 参与学习社群活动
   - 查看学习数据和分析报告

### 6.2 超级管理员
1. **登录**：使用超级管理员账号登录系统
2. **系统管理**：
   - 管理用户账号和权限
   - 管理学校信息
   - 配置系统参数
   - 查看系统数据报表
   - 监控系统操作日志

### 6.3 学校管理员
1. **登录**：使用学校管理员账号登录系统
2. **学校管理**：
   - 管理学校用户
   - 管理学校资源
   - 配置学校设置
   - 查看学校数据报表
   - 发布学校通知

## 8. 系统部署与维护

### 8.1 部署环境
- 前端：Node.js 14+
- 后端：Java 11+, MySQL 5.7+
- 服务器：Linux/Windows

### 8.2 部署步骤
1. **前端部署**：
   - 安装依赖：`npm install`
   - 构建项目：`npm run build`
   - 部署到Web服务器

2. **后端部署**：
   - 编译项目：`mvn clean package`
   - 部署jar包：`java -jar xueya-assistant.jar`
   - 配置数据库连接
